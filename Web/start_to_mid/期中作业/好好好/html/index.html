我的大学

　温州大学是浙南闽北赣东区域唯一的综合性大学、浙江省重点建设高校、博士学位授予单位，坐落于中国历史文化名城、改革开放先行区的温州市。学校由温州师范学院和原温州大学于2004年合并组建而成，已有90年办学历史，国家最高科学技术奖获得者谷超豪院士曾任校长。

我的学院

温州大学计算机与人工智能学院，是在新一轮科技革命和产业变革的时代背景下，顺应国家科技发展战略及区域产业人才培养需求而成立，其前身为温州师范学院计算机科学系与原温州大学信息科学与工程学院，有着良好的办学基础与历史沉淀。

21大数据1

21大数据1是一个团结友爱的集体，秉承“求学问是，敢为人先”的校训，紧跟时代步伐，聚焦劳动教育，在班主任和辅导员的带领下，努力以专业技能为载体，融劳动教育为全体成员成长成才的助推器。

个人 简介

我叫施政，是一名大三学生，就读于温州大学计算机与人工智能学院。我所在的班级是21数据科学与大数据技术1班。


在三年的学习生活中，我积极探索不同的编程语言和开发工具，包括Python、Java、C++等。 我熟练掌握编程语言的基本语法和常用库函数，并且能够使用它们进行软件开发和问题解决。 我还学习了各种算法和数据结构，了解它们的原理和应用场景，并且能够应用它们解决实际问题。 我热爱学习和不断探索新的技术和领域。我相信计算机科学和技术的发展将给社会带来巨大的变革和进步。


除了学业，我也热衷于探索不同的文化和旅行。我来自一个美丽的小城市，有着丰富的历史和文化遗产。 我热爱我的家乡，喜欢在个人站点上分享我的家乡的景点、民俗、美食和文化。 我希望通过这样的方式，让更多的人了解和欣赏我的家乡之美。


希望通过个人站点，与更多的人分享我的学习和生活经验。

我的家乡

诸葛八卦村

金华兰溪的诸葛八卦村是一个以诸葛亮为主题的旅游景区，游客可以了解诸葛亮的事迹和文化， 欣赏表演和参与体验活动，品尝当地美食和购买特产。这是一个融合了历史文化和旅游体验的地方

芝堰村建筑群

金华兰溪的芝堰村建筑群是一个保存完好的传统江南水乡村落，游客可以了解江南水乡的文化传统， 感受江南水乡的浪漫和宁静，品尝当地的美食，是一个融合了历史文化和旅游体验的地方

杨梅

金华兰溪是中国著名的杨梅产地之一，被誉为“中国杨梅之乡” 金华兰溪的杨梅以其鲜美的口感和独特的风味而闻名，游客可以品尝新鲜的杨梅， 了解杨梅的种植和加工过程，购买杨梅制品作为纪念品

兰庆鸡子馃

兰庆鸡子馃是金华兰溪地区的传统特色美食，以其酥脆的外表和嫩滑的内部而受到喜爱。 游客可以品尝这道美食，体验兰溪的独特风味和地方文化
